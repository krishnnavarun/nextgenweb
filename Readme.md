# NextGenWeb 🎓
> College Placement Opportunities Portal — MERN Stack

## Features
- Student, Company & Admin roles with JWT auth
- Browse & apply for jobs
- Company job postings & applicant management
- Admin dashboard — approve/reject
- Placement stats & charts
- Resume upload
- AI Chatbot (Claude API)

## Stack
- Frontend: React + Vite (latest) + Tailwind CSS
- Backend: Node.js + Express.js
- Database: MongoDB + Mongoose
- Auth: JWT + bcryptjs

## Quick Start

### 1. Clone & install
```bash
git clone <your-repo>
cd nextgenweb
npm run install-all
```

### 2. Setup env
```bash
cd server
cp .env.example .env
# Edit .env with your values
```

### 3. Run
```bash
cd ..
npm run dev
```
- Client → http://localhost:5173
- Server → http://localhost:5000
